# [Gold II] 디스크 트리 - 7432

[문제 링크](https://www.acmicpc.net/problem/7432)

### 문제 요약

<p> 중요한 디렉토리의 경로가 $N$개 주어진다. 이 정보를 통해 전체 디렉토리 구조를 출력해보자. </p>

### 제한

TL : 1sec, ML : 128 MB

$1 ≤ N ≤ 500$

### 성능 요약

메모리: 4804 KB, 시간: 12 ms

### 분류

자료 구조(data structures), 문자열(string), 트리(trees), 트라이(trie), 트리를 사용한 집합과 맵(tree_set)
